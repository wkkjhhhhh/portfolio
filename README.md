## 개요

앱설명, 사이트 링크 넣기

## 기능

게시판에 구현된 기능 넣기

- 메인 화면 : 판매하고자 하는 상품을 확인.
- 카테고리 : 판매등록한 상품들을 확인하고 관리.
- 관리자 페이지 : 상품 등록 / 상품 수정 / 카테고리 생성 / 회원관리.
